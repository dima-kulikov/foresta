
1 - npm i
2 - запускати всю махіну командою gulp

.pipe(plumber()) -- стоїть при зборці всіх елементів, не дає ломати проект і просто не дає працювати зломаній частині. 
При необхідності, просто закоментувати у всіх функцій в gulpfile.js щоб відключити.

щоби бачити помилки, встановленно npm i gulp-notify --save-dev


- Установлен автопрефиксер ("gulp-autoprefixer": "^8.0.0")
- для красивого читательного css файла (gulp-cssbeautify), при написании sass делает на выходе чистый css
- удаление комментариев при сборке (gulp-strip-css-comments)
- gulp-rename
- gulp-sass/sass (нужно ставить 2 пакета)
- gulp-cssnano убирает точки с запятыми и пробелы, делает цсс файл в одну строку
- gulp-rigger склеивает js файлы (делает с многих файлов при сборке в один большой)
- gulp-uglify робить мініфікатор файлів js
- gulp-plumber галп не ламається при помилках, а видає помилку
- gulp-imagemin зжимає і оптимізує зображення ("^7.1.0" стоїть більш стара версія, оскільки нові видають помилку)
- del стоїть більш стара версія, нові глюкаві
- panini для html взаємозвьязка
- browser-sync для оновлення сторінки браузера (по типу лайв сервера)
